# Swift 언어 심화

Swift 언어 자체의 깊은 이해를 위한 학습 공간입니다.
앨런강의에서 다루는 기초 문법이 아닌, **면접에서 깊이를 보여줄 수 있는 심화 내용**을 정리합니다.

## 폴더 구조

| 폴더 | 핵심 내용 | 면접 빈출도 |
|------|----------|------------|
| `arc-memory/` | ARC, 메모리 관리, Value vs Reference | ★★★★★ |
| `protocol-oriented/` | POP, Generics, Associated Type | ★★★★★ |
| `error-handling/` | Error, Result, do-catch, throws | ★★★ |
| `advanced/` | Opaque Type, Result Builder, Property Wrapper | ★★★ |

## 학습 방법
- Swift 공식 문서 (swift.org) 기반으로 정리
- Playground에서 실습 코드 작성 후 `.swift` 파일로 저장
- 마크다운에 개념 + 코드 예시 + 면접 Q&A 포함
