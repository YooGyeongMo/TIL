# 개발 도구

iOS 개발 실무에서 사용하는 도구들을 학습합니다.
57%의 타겟 회사가 Tuist/SPM 경험을 요구합니다.

## 폴더 구조

| 폴더 | 도구 | 요구 회사 | 학습 시기 |
|------|------|----------|----------|
| `tuist/` | Tuist (프로젝트 관리) | 현대차, 토스, 당근 | Week 4 |
| `spm/` | Swift Package Manager | 대부분 | Week 3 |
| `fastlane/` | Fastlane (CI/CD 자동화) | 오늘의집, 당근 | Week 4 |
| `github-actions/` | GitHub Actions (CI) | 공통 | Week 4 |

## 학습 방법
- 각 도구를 SUSA24 또는 Naru 프로젝트에 **실제 적용**하면서 학습
- 설정 과정과 트러블슈팅을 마크다운으로 기록
- 적용 후 GitHub에 커밋 → 포트폴리오 가치 상승

## 우선순위
1. SPM (이미 일부 사용중, 정리만 하면 됨)
2. Tuist (프로젝트 구조 개선)
3. GitHub Actions (CI 파이프라인)
4. Fastlane (배포 자동화)
