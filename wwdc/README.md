# WWDC 세션 정리

iOS 면접에서 "최신 기술 트렌드를 따라가는 개발자"임을 보여주는 핵심 영역.
모든 세션을 볼 필요 없이, **면접 빈출 세션**만 선별하여 정리합니다.

## 폴더 구조
```
wwdc/
├── 2024/          # WWDC24 핵심 세션
├── 2025/          # WWDC25 핵심 세션
└── essential-sessions.md  # 면접 필수 세션 리스트
```

## 정리 방식
각 세션을 마크다운 파일로 정리:
```markdown
# [세션 제목]

## 한줄 요약
## 핵심 내용 (3~5개 포인트)
## 코드 예시
## 면접에서 활용할 포인트
## 관련 TIL 링크
```

## 학습 방법
1. `essential-sessions.md`에서 우선순위 높은 세션부터 시청
2. 1.5배속으로 시청, 핵심만 마크다운 정리
3. 하루 1세션 (30분~1시간)
4. 해당 토픽 폴더에 연결 (예: Concurrency 세션 → `swift/concurrency/`에도 링크)

## 시청 팁
- Apple Developer 앱 또는 developer.apple.com/videos 에서 시청
- 자막 활용 (영어 자막 → 영어 공부도 겸)
- 코드 예시는 반드시 Playground에서 직접 실행
