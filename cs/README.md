# CS 기초 (면접 대비)

iOS 기술면접에서 반드시 나오는 CS 기초 지식을 정리합니다.

## 학습 목표
- 기술면접에서 자신있게 답변할 수 있는 수준
- 단순 암기가 아닌 **iOS 개발과 연결지어** 설명할 수 있는 수준

## 폴더 구조

| 폴더 | 주제 | 면접 빈출도 |
|------|------|------------|
| `data-structures/` | 자료구조 (Array, LinkedList, Tree, Hash 등) | ★★★★★ |
| `operating-system/` | OS (프로세스, 스레드, 메모리, 동기화) | ★★★★★ |
| `network/` | 네트워크 (HTTP, TCP/IP, REST, 소켓) | ★★★★ |
| `design-patterns/` | 디자인 패턴 (Singleton, Observer, Factory 등) | ★★★★ |

## 학습 방법
1. 각 토픽을 마크다운으로 **개념 + iOS 적용 예시** 형태로 정리
2. 면접 예상 질문과 답변을 함께 작성
3. 하루 30분~1시간, 1토픽씩 정리
